CSS Functions

Definição

Valor seguido por parênteses

Argumentos são calculados dentro dos parênteses

Pedaços de códigos que podem ser reusados

Importante para responsividade do layout

calc()

Diferentes unidades podem ser calculadas

width: calc(60% + 30px);

min()

O menor valor é aplicado

Em dispositivos menores o valor será de 70%, pois será menor que 300px

width: min(300px, 70%);

max()

O maior valor é aplicado

Em dispositivos maiores o valor será de 40%, pois será maior que 200px

width: max(200px, 40%);

clamp()

(menor, ideal, maior)

"Quando alcançar certo valor, o mesmo será aplicado."

width: clamp(220px, 50%, 80vw);
